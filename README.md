# RescueEye_Sim2Real

## 조난자 데이터셋을 YOLOv5를 이용하여 학습하고, Objects detection과 데이터 증강을 통해 모델 성능 개선.
- 효과적인 Data Augmentation 찾기
- Detection에 실패한 이미지에 대해 CAM 분석을 통해 설명 가능한 AI


> SARD 약 4.5GB
> -이미지 크기 : 1920*1080
> -레이블 : 사람(Person)


> PABLO 합성 데이터 약 45GB
> -이미지 크기 : 3840*2160
> -레이블 : Person, Pad, Pablo Mark
